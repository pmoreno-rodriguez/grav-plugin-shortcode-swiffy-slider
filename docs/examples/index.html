Image carousel with infinite sliding

Simple image carousel with 'infinite sliding'. Using a bit of scripting, this example will look like it has infinite scrolling by going to the first slide when the last has been reached. The trick is to repeat the first slide as the last and then use JS to go back to the first when the end is reached. This only works when showing one slide at a time and if used with reveal will cause jumping

Full screen image carousel

An image slider put inside a modal (In this case a full screen Bootstrap modal).

Javascript is used to set focus on the first slide when the modal is opened and the slides have tabindex="-1 " to support keyboard navigation

Click the image to see the carousel in full width and navigate with keyboard arrows

(Press ESC to close fullscreen view - or tap the image)
