Slider sizes

Options affecting the number of slides, their sizes and presentation using .slider-item-show* options.

There can be either 1, 2, 3, 4, 5 or 6 slides per page of the slider.

By adding .slider-item-reveal the next and previous slide is revealed - when sliding the visible slides will be centered.

When using .slider-item-reveal it is also possible to add .slider-item-snapstart. Using that option will snap the slides to the start so only the next slide is revealed and not the previous. This setting only has a visible effect when using the .slider-item-reveal

On device sizes with a width below 992px, configurations with 1-4 slides changes to 1 slide per page. 6 slides changes to 4 slides and then 2 slides per page in mobile view.

Also .slider-item-snapstart and .slider-item-nogap are used as they affect how the slides behaves
